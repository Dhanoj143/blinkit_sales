# blinkit_sales

Blinkit Sales and Operations Analysis in Power BI

The Blinkit Analysis project uses Power BI to explore key metrics around sales, delivery efficiency, and customer behavior for a rapid grocery delivery service. This dashboard visualizes critical KPIs, such as total revenue, order frequency, top-selling products, and average delivery time. By analyzing data on customer locations, purchase patterns, and order volumes, the project helps identify peak demand times and popular product categories. Operational insights like delivery time trends, delivery agent efficiency, and fulfillment rates provide actionable insights for enhancing service quality.
